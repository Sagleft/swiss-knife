# swiss-knife

Library with all sorts of things to create code prototypes super fast

## install

```bash
go get github.com/Sagleft/swiss-knife
```

### usage example

```go
package main

import (
    "fmt"
    swissknife "github.com/Sagleft/swiss-knife"
)

func main() {
    fmt.Println("Hello, World!")
    swissknife.RunInBackground()
}
```
